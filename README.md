```markdown
# 项目简介
本项目是一款基于 Flask 开发的岗位招聘数据可视化工具。

## 核心功能
- 数据读取：支持从文件中读取岗位招聘数据。
- 可视化图表：利用 Pyecharts 库生成各类可视化图表。

## 适用场景
- 人力资源部门：用于分析岗位招聘趋势。
- 招聘公司：辅助招聘决策，优化招聘策略。
- 数据分析师：研究岗位招聘市场动态。

## 技术特点
- 基于 Flask 框架开发，易于部署和扩展。
- Pyecharts 库支持多种图表类型，可视化效果丰富。
```
